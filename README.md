# star-manager
A tool for Github star management.

## Features
- [cssnext](http://cssnext.io/) syntax support
- React + Mobx
- HMR support

## How to start develop?
1. `git@github.com:tomoyuen/star-manager.git`
2. `cd star-manager && npm install`
3. `npm start`

**Want to start electron console?**

run `npm run debug`
